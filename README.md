Animal Kingdom Africa

Animal Kingdom Africa is a front-end website that showcases different animals found in Africa. The site is built in React and is fully responsive, making it easy to view on different devices. Features

    Displays information about different animals found in Africa
    Has beautiful animations to engage users
    The design is slick and clean, making it easy to navigate
    Fully responsive, ensuring optimal viewing on all devices

Installation

    Clone the repository to your local machine
    Run npm install to install the dependencies
    Run npm start to start the development server

Usage

Visit the deployed site [HERE](https://taupe-figolla-2e3faf.netlify.app/) to view the Animal Kingdom Africa website. Contributing

If you'd like to contribute to this project, please fork the repository and create a pull request with your changes. Credits
